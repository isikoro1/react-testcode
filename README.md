# react-testcode

React をとにかく書いて慣れるための練習用リポジトリです。  
Vite + React の最小構成に、すぐ触れるサンプルを 3 つ置いてあります。

## Setup

```bash
npm install
npm run dev
```

## Samples

- `src/examples/CounterLab.jsx`
  `useState` とクリックイベントの基本。
- `src/examples/PaletteLab.jsx`
  controlled input と派生 UI の基本。
- `src/examples/TaskBoardLab.jsx`
  配列 state、追加、切り替え、filter の基本。

## Build

```bash
npm run build
```
