# react-testcode

React と TypeScript をとにかく書いて慣れるための練習用リポジトリです。  
Vite + React + TypeScript の最小構成に、すぐ触れるサンプルを置いてあります。

## Setup

```bash
npm install
npm run dev
npm run typecheck
```

## Samples

- `src/examples/CounterLab.tsx`
  `useState` とクリックイベントの基本。
- `src/examples/PaletteLab.tsx`
  controlled input と派生 UI の基本。
- `src/examples/TaskBoardLab.tsx`
  配列 state、追加、切り替え、filter の基本。
- `src/examples/TimerLab.tsx`
  `useEffect` とクリーンアップの基本。
